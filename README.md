線上投票系統

使用以下項目實作本系統:
1.Spring Boot Thymeleaf 和 Hibernate
2.關聯式資料庫:PostgreSQL
3.以RESTful API 取得投票者、投票項目、投票紀錄
4.使用Maven作為專案工具
5.SQL:DDL、DML，請參考/DB資料夾

運作Spring Boot，由localhost:8080，index.html顯示線上投票系統。

尚未實作:

2.資料庫部分:Stored Procedure、Transaction、SQL Injection
